# 🚀 Your Site PRO — Універсальний сучасний лендінг-пейдж шаблон

**Версія 1.0 (28 лютого 2026)**  
**Автор:** Ruslan Bilohash | [bilohash.com](https://bilohash.com)

Сучасний, швидкий, повністю керований через адмінку односторінковий сайт (landing page).  
Ідеально підходить для **будь-якого бізнесу**: послуги, продажі, консультації, портфоліо, ресторани, салони, автосервіси, юридичні послуги, IT-компанії тощо.

## ✨ Основні можливості версії 1.0

### ✅ Повністю керований сайт без коду
- Всі тексти, зображення, меню, відгуки, FAQ, SEO — редагуються в зручній адмінці
- Дані зберігаються в одному файлі `content.json`
- Одна кнопка «ЗБЕРЕГТИ ВСІ ЗМІНИ»

### ✅ Потужна адмінка (`admin.php`)
- Захист паролем (змінюється в коді)
- Кнопка **«Вихід»** (мобільна + десктоп)
- 15 вкладок для повного контролю:
  - Загальні налаштування
  - Шапка сайту (логотип + редаговане меню)
  - Meta Tags
  - OG + Twitter Cards
  - Hero-секція
  - Послуги (картки з фото)
  - Переваги
  - Як ми працюємо (кроки)
  - Галерея
  - Відгуки
  - SEO текст
  - FAQ
  - Форма замовлення / заявки
  - Footer + Schema.org
  - **Налаштування Email** (отримувач, відправник, тема)
  - Custom Code (head, CSS, JS, footer HTML)

### ✅ Форма заявки + красивий Email
- `submit.php` — сучасний обробник з валідацією
- Гарний HTML-лист (градієнт, іконки, адаптивний)
- Автоматичний редирект з урахуванням мовних версій
- Налаштування email повністю через адмінку
- Захист від порожніх полів

### ✅ Сторінка помилки 404
- Стильна сторінка `404.php` з таймером 8 секунд
- Автоматична переадресація на головну
- `.htaccess` для правильної обробки всіх 404

### ✅ Дизайн та технічні переваги
- Повністю адаптивний (мобільні + десктоп)
- Швидке завантаження (Tailwind CDN)
- Schema.org LocalBusiness (для SEO)
- Open Graph + Twitter Cards
- Підтримка кількох мов (UA + готові посилання на EN, RU, LT, NO)
- Мобільне меню + дропдаун мов
- Анімації hover
- Кнопка «Адмінпанель» в футері (прихована для клієнтів)

### ✅ Безпека та зручність
- Вихід з адмінки одним кліком
- Авто-очищення таймера
- Всі дані екрануються
- Легко розгортається на будь-якому хостингу з PHP 7.4+

## 🚀 Як встановити за 2 хвилини

1. Завантаж всі файли на хостинг
2. Перейди за адресою `/admin.php`
3. Введи пароль `admin123` (зміни в `admin.php` на свій)
4. Налаштуй сайт у адмінці → Натисни «ЗБЕРЕГТИ ВСІ ЗМІНИ»
5. Готово! Сайт працює

**Для 404:** `.htaccess` вже налаштований.

---

## 🔑 Що змінити перед запуском

- Пароль в `admin.php`
- Email отримувача в адмінці → вкладка «Налаштування Email»
- Всі плейсхолдери `[МІСТО]`, `yourdomain.com`, `Your site!`
- Логотип, фото, тексти під свій бізнес
- Додай свої реальні зображення замість picsum.photos
