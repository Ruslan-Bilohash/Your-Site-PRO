# 🚀 Your Site PRO — Universal Modern Landing Page Template
**Version 1.0 (February 28, 2026)**
**Author:** Ruslan Bilohash | [bilohash.com](https://bilohash.com)

A modern, fast, fully manageable single-page website (landing page) through the admin panel.

Perfect for **any business**: services, sales, consultations, portfolio, restaurants, salons, auto services, legal services, IT companies, etc.

---
## ✨ Key Features of Version 1.0
### ✅ Fully manageable website — no coding required
- All texts, images, menu, reviews, FAQ, SEO — editable in a convenient admin panel
- Data is stored in a single file `content.json`
- One button **“SAVE ALL CHANGES”**

### ✅ Powerful admin panel (`admin.php`)
- Password protection (changeable in the code)
- **“Logout”** button (mobile + desktop)
- 15 tabs for full control:
  - General settings
  - Site header (logo + editable menu)
  - Meta Tags
  - OG + Twitter Cards
  - Hero section
  - Services (cards with photos)
  - Advantages / Benefits
  - How we work (steps)
  - Gallery
  - Reviews / Testimonials
  - SEO text
  - FAQ
  - Order / Request form
  - Footer + Schema.org
  - **Email settings** (recipient, sender, subject)
  - Custom Code (head, CSS, JS, footer HTML)

### ✅ Request form + beautiful email
- `submit.php` — modern handler with validation
- Nice HTML email (gradient, icons, responsive)
- Automatic redirect considering language versions
- Full email configuration via admin panel
- Protection against empty fields

### ✅ 404 error page
- Stylish `404.php` page with 8-second timer
- Automatic redirect to homepage
- `.htaccess` already configured for proper 404 handling

### ✅ Design & Technical Advantages
- Fully responsive (mobile + desktop)
- Fast loading (Tailwind CDN)
- Schema.org LocalBusiness (for SEO)
- Open Graph + Twitter Cards
- Multi-language support (UA + ready links for EN, RU, LT, NO)
- Mobile menu + language dropdown
- Hover animations
- “Admin Panel” button in footer (hidden for clients)

### ✅ Security & Convenience
- One-click logout from admin
- Auto-clearing of timer
- All data is escaped
- Easy deployment on any hosting with PHP 7.4+

## 🚀 Installation in 2 Minutes
1. Upload all files to your hosting
2. Go to `/admin.php`
3. Enter password `admin123` (change it in `admin.php` to your own)
4. Configure the site in the admin panel → Click “SAVE ALL CHANGES”
5. Done! The site is live

**For 404:** `.htaccess` is already configured.

---
## 🔑 What to Change Before Launch
- Password in `admin.php`
- Email recipient in admin panel → “Email Settings” tab
- All placeholders `[CITY]`, `yourdomain.com`, `Your site!`
- Logo, photos, texts — customize for your business
- Replace placeholder images (picsum.photos) with your real ones
